Valentine's Day Page for Devyani

This is a romantic, interactive Valentine's Day webpage with floating hearts, falling petals, a message reveal, and a flip-card note.

How to view
- Open index.html in a browser.

Features
- Floating hearts animation and gentle sparkle background.
- Tap-to-reveal message with soft fade and scale.
- 3D flip-card with a personal note.
- Floral bouquet section and smooth scroll transitions.
